使用帮助

1、命令查看帮助
jms-inspect_cli -h

2、使用命令
jms-inspect_cli --jms-config <file> --inspect-config <file> --machine-template <file>

#####################
编译脚本
# 环境 CentOS8.2 编译正常，其他环境可能会缺少依赖环境而失败
1、进入到requirements目录下
2、执行 compile.sh 即可
3、如果编译的宿主机已经编译过脚本，执行 compile.sh -c 即可
